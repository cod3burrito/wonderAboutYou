# wonderAboutYou
A simple web app to help you find out more about your friends

## How to use
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open your Insomnia and stuff

## Technologies Used
* JavaScript
* Node.js
* Express.js
* MongoDB
* Mongoose
* Moment.js
